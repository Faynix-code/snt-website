Un site Static mais pourquoi ?

Static ?

Un site internet statique, également appelé site web statique, est un type de site web dont le contenu est fixe et n'est pas généré dynamiquement à la demande de l'utilisateur. Contrairement aux sites web dynamiques qui extraient des données de bases de données et les génèrent à la volée, les sites statiques affichent des pages préconçues qui ne changent pas en fonction des interactions de l'utilisateur.
